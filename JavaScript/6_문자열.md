## 문자열

자바스크립트에서는 문자가 없고 다 문자열로 처리합니다.

'a', "a", 'string', "string" 모두 문자열입니다.

또한 문자열을 객체로 인식합니다.

```
"ab:cd".split(":");         //["ab","cd"]
"ab:cd".replace(":","$");   //"ab$cd"
```
